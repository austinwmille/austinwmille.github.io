Here's what you need to do: (use the checkboxes to mark your progress)

  1. Open minecraft and log in and stuff.


  2. Go to the "Server" tab, not "Friends" like we were doing before.


  3. Scroll down, past the 'featured' servers, and press add server.

    • If you cannot see that button then you are probably using a console like a Switch or Xbox. You can still connect by changing your internet settings, and don't worry it's not very hard.

    • There are additional instructions for you at the bottom of this webpage. Hit me up with any questions!


  4. The server name is "nipsded". I do not know if this is important yet. You can try to name it whatever you want..i think it'll probs work and anyways.


  5. The server address is "minecraft.austinmiller.net". Notice that it's different from the page you're on right now!
    • It might just say address, or IP address, or something else. They are all the same thing!


  6. The port should be filled in for you already. It should say "19132". If it says a different number then change it. If it doesn't say anything at all then type it in.


  7. Now press Save! After saving the server details, it should take you back to the list of featured servers (which is stupid) but if you scroll down you should see "nipsded" (or whatever server name you gave it) and it should now be saved for whenever you want to join. Godspeed
